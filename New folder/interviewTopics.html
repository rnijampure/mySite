Question 1. What are the different JavaScript types?
Answer: Types are the building blocks of any language. JavaScript's types are:
  • a) Number
  • b) String,
  • c) Boolean,
  • d) Function,
  • e) Object,
  • f) Null,
  • g) Undefined.
  • Dates and Regular Expressions, and Arrays are all built-in objects. A function is also an object
Question 2.How can you convert a string to an integer (number in general) using javascript?
Answer: Javascript has a built-in parseInt() function which can be used to this. This function Parses a string argument and returns an integer of the specified radix or base.
Example: parseInt("23")

Question 5. What is the difference between a == and ===?
Answer: JavaScript has both strict and Type–converting (abstract) comparisons. A strict comparison (e.g. ===) is only true if the operands are the same Type. The more commonly used abstract comparison (e.g. ==) converts the operands to the same Type before making the comparison.
